RenameStickerSetRequest

Both users and bots may be able to use this request. See code examples.

---functions---
stickers.renameStickerSet#124b1c00 stickerset:InputStickerSet title:string = messages.StickerSet

Returns

messages.StickerSet

This type can be an instance of either:

StickerSetStickerSetNotModified

Parameters

stickersetInputStickerSet
titlestring

Known RPC errors

This request can't cause any RPC error as far as we know.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.stickers.RenameStickerSetRequest(
        stickerset=types.InputStickerSetID(
            id=-12398745604826,
            access_hash=-12398745604826
        ),
        title='My awesome title'
    ))
    print(result.stringify())
